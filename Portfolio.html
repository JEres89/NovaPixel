1. RESTful API with ASP.NET Core

    Developed a robust and scalable RESTful API using ASP.NET Core. The API serves as the backend for a fitness tracking application, handling user authentication, workout logging, and progress tracking. Leveraging Entity Framework Core, the API efficiently manages data across multiple tables, ensuring fast and secure transactions.

    Key Highlights

    • Implemented JWT-based authentication for secure access.
    • Utilized Swagger for comprehensive API documentation.
    • Integrated Azure Application Insights for monitoring and performance analysis.

  2. WPF Desktop Application - Inventory Manager

    Designed and developed a desktop application using WPF for small business inventory management. The application enables users to track stock levels, update product details, and generate comprehensive sales reports. It features an intuitive UI with data visualization charts for better insights.

    Key Highlights

    • Integrated MVVM architecture for clean and maintainable code.
    • Added real-time data synchronization with a local SQL database.
    • Incorporated drag-and-drop functionality for efficient item management.

  3. Frontend Web-store - Fashion Boutique

    Created an engaging and modern web-store interface for an online fashion boutique. The design emphasizes product visuals and seamless navigation, featuring dynamic filters, animations, and user-friendly layouts. Built using HTML5, CSS3, and JavaScript frameworks like React for optimal performance.

    Key Highlights

    • Implemented responsive design for consistent performance across devices.
    • Enhanced UX with interactive product carousels and dynamic search.
    • Integrated payment gateway APIs for secure transactions.
